This package includes the MATLAB code of the Non-convex Multi-Instance Learning algorithm WLRM, 
which solves the Functional Annotation problems for Human Protein Coding Isoforms. If you use our 
code, please cite our paper:

Luo, T., Zhang, W., Qiu, S., Yang, Y., Yi, D., Wang, G., ... & Wang, J. (2017, August). 
  Functional Annotation of Human Protein Coding Isoforms via Non-convex Multi-Instance Learning.
  In Proceedings of the 23rd ACM SIGKDD International Conference on Knowledge Discovery 
  and Data Mining (pp. 345-354). ACM.
